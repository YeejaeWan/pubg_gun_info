# pubg_gun_info

플러터 공부하면서 만드는 배틀그라운드 총기 정보 제공 어플리케이션

## 앱 소개
- AR, SR, DMR, SG, SMG, Pistol, LMG, 기타, 투척무기, 근접무기 정보를 확인할 수 있음.
- 데미지 측정을 통해 헬멧과 갑옷을 적용시켜 무기의 데미지 수치를 확인할 수 있음
